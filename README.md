# English Exam Prep Platform

## Назначение проекта
Образовательная платформа для самостоятельной подготовки к экзаменам по английскому языку. Помогает пользователям структурировать подготовку, отслеживать прогресс и выявлять слабые места.

## Состав репозитория
- docs/ — проектная документация, требования, постановка задачи;
- src/ — исходный код платформы;
- data/sample/ — демо-версия тестовых заданий;
- experiments/ — журнал тестирования и экспериментов;
- assets/ — изображения, схемы, макеты интерфейса;
- sql/ — скрипты для базы данных;
- tests/ — автоматические тесты;
- notebooks/ — Jupyter notebooks для анализа данных.

## План выполнения
1. Анализ существующих платформ и формулировка требований.
2. Проектирование архитектуры и базы данных.
3. Разработка прототипа (регистрация, тестирование, прогресс).
4. Наполнение контентом (тестовые задания по разделам).
5. Тестирование и сбор обратной связи.
6. Подготовка пояснительной записки.

## Текущий статус
Сформирована структура репозитория, создан паспорт проекта, начата постановка задачи.
